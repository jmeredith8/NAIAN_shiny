# NAIAN_shiny
Date: 6/15/26

Created by: Jakob Meredith (under the supervision of Dr. Sadie Ryan)

Shiny App showing the spatiotemporal trends of NAIAN insect ecology research.

About: This is a map dashboard showing the results of spatial insect ecology research conducted by Dr. Peter Dunn and his team for their publication "Extensive regional variation in the phenology of insects and their response to temperature across North America" in the journal Ecology. It shows the total biomass of insect samples caught at various sites in relation to different environmental variables.

Instructions: Download the CSV file, the R script file, and the R project file all in the same folder. Then, run the R script with the provided packages and libraries on your local computer.
